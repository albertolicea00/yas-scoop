# Design System - Yet Another Store for Scoop

This document outlines the visual identity and styling guidelines for **Yet Another Store for Scoop**.

## Color Palette

The design is built around a modern, premium Dark Theme featuring a deep store base color combined with a signature Teal accent.

### Color Tokens

- **Store Base Color (Dark Gray)**:
  - Hex: `#222629`
  - HSL: `hsl(206, 9%, 15%)`
- **Accent Color (Teal)**:
  - Hex: `#008080`
  - HSL: `hsl(180, 100%, 25%)`
- **Subtle Highlight**:
  - Hex: `#0080801A` (10% opacity)
- **Text Primary**:
  - Hex: `#F8F8F2`
- **Text Secondary**:
  - Hex: `#ACADAD`

## Visual Assets & Logo Concept

- **Base Structure**: Dark gray store icon outline or 3D block representing the storefront.
- **Accent Elements**: Highlights, borders, or light beams in **Teal**.
- **Inside the Store**: Stacked package cubes representing the packages managed by Scoop.
- **App Tag**: SCOOP

## Typography

- **Headings**: Outfit or Inter (sans-serif)
- **Body Text**: Inter (sans-serif)
- **Monospace (for CLI output/commands)**: Fira Code or JetBrains Mono
